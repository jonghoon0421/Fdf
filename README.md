# Fdf
**To install:**
* `git clone https://github.com/jonghoon0421/Fdf.git`
* `make`

**To execute:**
* `Usage: ./fdf [map]`

  * `[+, -] | Zoom in/out`
  * `[Q, W] | Rotate X-axis`
  * `[A, S] | Rotate Y-axis`
  * `[Z, X] | Rotate Z-axis`
  * `[O, P] | Change altitudes`
  * `[N]	| Reset to initial state (black screen before zoom-in)`
  * `[ESC]	| Exit`
  * ` Use the arrow keys to move the object in 4 directions`
